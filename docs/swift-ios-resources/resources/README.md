# Free Swift & iOS Web Resources

These are all **free to read** on the publisher's site. They stay as live URLs
on purpose — Swift evolves, the books stay current, and downloading snapshots
goes stale within weeks. Each link has a one-line description of why it's
worth your time.

## Apple-official (the canonical references)

- **[Apple Developer Documentation](https://developer.apple.com/documentation/)** — every public API, every framework, every platform. Always current. Search is excellent.
- **[The Swift Programming Language (online)](https://docs.swift.org/swift-book/documentation/the-swift-programming-language/)** — the bundled source mirrored as a hosted DocC site. Read here if you don't want the bundle.
- **[Swift.org](https://swift.org/)** — Swift's home. Release notes, evolution dashboard, getting-started guides.
- **[Swift Standard Library](https://developer.apple.com/documentation/swift)** — every type, every method, searchable.
- **[Apple Sample Code](https://developer.apple.com/sample-code/)** — Apple's official sample projects. Always up to date with the latest SDKs.
- **[WWDC Videos](https://developer.apple.com/videos/)** — every WWDC session since 2014. Free. Most have transcripts.
- **[Swift Forums](https://forums.swift.org/)** — the official Swift community forum. Where the language design happens.
- **[Hacking with Swift +100 Days of SwiftUI](https://www.hackingwithswift.com/100)** — Paul Hudson's free, structured SwiftUI curriculum. Highly recommended for beginners.
- **[Apple's "Develop in Swift" tutorials](https://developer.apple.com/tutorials/develop-in-swift-tutorials)** — Apple's own structured learning path.

## Free articles & deep dives (high quality, not free books)

- **[objc.io](https://www.objc.io/)** — long-form articles on Swift, SwiftUI, Combine, architecture. Some free, some paid. The free ones are excellent.
- **[Point-Free](https://www.pointfree.co/)** — Brandon Williams & Stephen Celis on functional programming and Swift architecture. Some free episodes.
- **[Swift by Sundell](https://www.swiftbysundell.com/)** — John Sundell's weekly articles. Free, deep, and extensive archive.
- **[SwiftLee](https://www.avanderlee.com/)** — Antoine van der Lee's Swift tips. Weekly, practical, well-written.
- **[Donny Wals](https://www.donnywals.com/)** — Combine, Core Data, SwiftUI deep dives.
- **[Fatbobman Blog](https://fatbobman.com/)** — SwiftData, SwiftUI, Core Data. Particularly good for Apple's newer frameworks.
- **[Keith Harrison's Using SwiftUI](https://useyourloaf.com/blog/)** — long-running iOS dev blog, SwiftUI focus.
- **[Ole Begemann](https://oleb.net/blog/)** — Swift and Apple platform internals.

## Tools & references

- **[SwiftLint](https://github.com/realm/SwiftLint)** — the de-facto Swift style enforcer. Free, open source.
- **[SwiftFormat](https://github.com/nicklockwood/SwiftFormat)** — code formatter, the other half of the SwiftLint/SwiftFormat combo. By Nick Lockwood.
- **[swift-book-pdf](https://github.com/Swift-Book-CN/swift-book-pdf)** — community tool that converts the bundled TSPL source into PDF/EPUB for offline reading.
- **[Swift Package Index](https://swiftpackageindex.com/)** — searchable index of every Swift package on GitHub. The place to find a library.
- **[Dash](https://kapeli.com/dash)** — offline API documentation browser. One-time purchase, free trial. macOS only.
- **[Iconify](https://iconify.design/)** — free icon search, used widely in SwiftUI projects.

## Free interactive learning

- **[Swift Playground](https://developer.apple.com/swift-playgrounds/)** — official iPad/Mac app for learning Swift interactively.
- **[Exercism Swift Track](https://exercism.org/tracks/swift)** — free mentored coding exercises. Good for working through idiomatic Swift patterns.
- **[Swift Koans](https://github.com/tnantoka/swift_koans)** — Ruby-style incremental exercises. Japanese origin, English text.
- **[SwiftUI Lab](https://swiftui-lab.com/)** — Javier Torrente's advanced SwiftUI experiments (custom layouts, geometry, etc.).
- **[Fucking SwiftUI](https://fuckingswiftui.com/)** — searchable, opinionated reference of SwiftUI gotchas. Crude name, solid content.

## For the architecture / testing side

- **[Point-Free's Composable Architecture](https://github.com/pointfreeco/swift-composable-architecture)** — the most-discussed state-management library for SwiftUI. Free, open source.
- **[The Book of SwiftUI](https://thebookofswiftui.com/)** — free online book by Matt Vandusen, opinionated and practical.
- **[Unit Testing in Swift](https://www.mokacoding.com/blog/swift-unit-testing-resources/)** — curated list of free testing resources, by Gio Lodi.

## Newsletters (free, weekly or so)

- **[Swift Weekly Brief](https://swiftweeklybrief.com/)** — curated by Jesse Squires, the de-facto Swift newsletter.
- **[iOS Dev Weekly](https://iosdevweekly.com/)** — Dave Verwer, broader iOS ecosystem.
- **[Indie iOS Dev Weekly](https://indieiosdevweekly.com/)** — focus on indie developers and shipping.

## YouTube channels

- **[Swiftful Thinking](https://www.youtube.com/@SwiftfulThinking)** — Nick Sarno's SwiftUI courses, structured and free.
- **[Sean Allen](https://www.youtube.com/@SeanAllen)** — full iOS dev curriculum, beginner-friendly.
- **[Kavsoft](https://www.youtube.com/@Kavsoft)** — SwiftUI animations and visual effects, very practical.
- **[Stewart Lynch](https://www.youtube.com/@StewartLynch)** — Swift, SwiftUI, Combine, very methodical.
- **[Code with Chris](https://www.youtube.com/@CodeWithChris)** — beginner-focused, full projects.
